Detailed Description

template<typename Seq>
struct coveo::seq_traits< Seq >

Traits class containing definitions pertaining to a sequence. A shorthand for coveo::seq_element_traits that infers the sequence's value_type from the return value of its iterators. Also provides the type of iterator used by the sequence.
